Dashboard 3: Regional Analysis: Insights by State

Purpose

This dashboard provides a detailed regional analysis by state, focusing on Total Revenue and Average Order Value (AOV) metrics. It empowers users to uncover geographical trends, identify top-performing states, and gain deeper insights into state-level and city-level performance.

Key Insights

Regional Trends

  • Total Revenue: The southeast region dominated revenue share across 2016–2018, contributing 73.67% of total revenue. São Paulo led with 37.33% of overall revenue.
  • Average Order Value (AOV): No strong regional clustering was observed. Top AOV states were scattered across the northeast and northwest regions, including Acre (AC).

Interactive Features

  • Filters and toggles provide dynamic updates across all visuals.
  • Visual insights update titles and callouts based on the selected metric (Total Revenue or AOV) and time periods.

Dashboard Features and Sections

Filters and Selections

Time Slicer: Filters data by year, quarter, or month, ensuring all metrics and visuals dynamically update.
Selected Metric Toggle: Allows toggling between Total Revenue and Average Order Value (AOV). Visualizations and callouts dynamically reflect the chosen metric.
Revenue % KPI Card: Displays revenue contribution as a percentage of the overall dataset. Updates based on filters and selections.

Visualizations

1. Top 5 Bar Chart

Purpose: Displays the top 5 states based on the selected metric (Total Revenue or AOV).
Features:

  • Title dynamically updates based on the selected metric.
  • Includes a callout showing the top state by revenue share and the top state by AOV.
  • Filters by time slicer to display insights for specific years or quarters.

2. Top 5 Scatter Chart

Purpose: Provides the same insights as the bar chart but includes a size element for order count.
Features:

  • Dynamically displays top states based on the selected metric.
  • Bubble sizes reflect order volumes for easy comparison.

Screenshots

Top 5 Scatter Chart

3. State-Level Revenue and Order Metrics (Matrix Table)

Purpose: Displays granular insights at the state and city levels.
Features:

  • Columns include: State (drillable to city), Selected Metric, Revenue % per year, and Order Count.
  • Revenue % normalizes annually to show contributions within selected years.
  • Dynamic sorting by the selected metric ensures top-performing states appear first.

4. Geographical Distribution Map

Purpose: Visualizes the distribution of the selected metric across all states.
Features:
